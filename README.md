# IPL Crunch '26 – Data Analytics Project

## Overview
This project analyzes IPL match data to identify winning patterns, batting trends, bowling impact, toss influence, and match-phase performance.

## Project Objective
Analyze IPL match data across five seasons to identify
winning patterns, batting and bowling performance,
toss impact, and match-phase trends using data analytics.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab
- Canva

## Key Insights
- Middle overs contributed the highest cumulative runs.
- Aggressive strike rates improved winning chances.
- Dot-ball pressure increased wicket opportunities.
- Winning the toss alone did not guarantee victory.
